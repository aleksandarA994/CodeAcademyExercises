Ајде да направиме основна HTML страница **

Секоја веб страница се состои од:

  • HTML
  • CSS
  • Javascript

HTML е јазик за означување и дефинирање веб страница.

CSS е јазик кој се употребува за да ја стилизира содржината (изгледот и форматирањето) на документ напишан во јазик за означување.

Што е Javascript ?

Видео за Javascript тука.

Правила за пишување HTML код:

  • Сите елементи и нивните својства се пишуваат со мали букви
  • Сите вредности на атрибутите на елементите треба да се во двојни наводници
  • Сите елементи треба да бидат затворени
