Кратко објаснување за Javascript:

JavaScript е најпопуларниот и најшироко распространет јазик за претежниот дел веб проекти. Станува збор за client-side програмски јазик, односно софтвер кој работи на страната на клиентот. Тоа всушност значи дека програмскиот код е испратен кон компјутерот на корисникот со остатокот од кодот од веб страната а потоа прелистувачот го чита тој код и прави тоа што му е наложено.

На овој линк можете да погледнете видео за Javascript.
